Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. Latitude and longitude were created by a random set of latitude and longitude combinations using Python. And then, we identified nearest city for each latitude and longitude combination using 'citipy'.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. We also conducted correlation and regression analyses to understand the relatioship between the latitude and temperature relationship. We conducted the analyses separatelly for the Northern and Southern Hemisphere. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
